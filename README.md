# Physico-chemical methods and properties

Description: An ontology of physico-chemical methods and properties.

More information can be found at http://obofoundry.org/ontology/fix

## Versions

### Stable release versions

The latest version of the ontology can always be found at:

http://purl.obolibrary.org/obo/fix.owl

(note this will not show up until the request has been approved by obofoundry.org)

### Editors' version

Editors of this ontology should use the edit version, [src/ontology/fix-edit.owl](src/ontology/fix-edit.obo)
