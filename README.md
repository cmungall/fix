# Physico-chemical methods and properties

This repository is a rescue home for the orphaned ontology `fix`.

It was created in March 2026 to keep FIX available in a durable GitHub/ODK repository for archival, indexing, and mapping use.

Chris Mungall (`cmungall`) is providing repository hosting and release automation only. He is not the original author, owner, or scientific maintainer of FIX.

Description: An ontology of physico-chemical methods and properties.

Original ontology page: http://obofoundry.org/ontology/fix

## Repository Files

- Rescued editable source: `src/ontology/fix-edit.obo`
- Main release artefacts committed to this repo: `fix.owl`, `fix.obo`
- Additional derived artefacts: `fix-full.owl`, `fix-full.obo`, `fix-base.owl`, `fix-base.obo`
- Preserved rescue snapshot: `src/ontology/upstream/fix-2020-04-13.owl`
